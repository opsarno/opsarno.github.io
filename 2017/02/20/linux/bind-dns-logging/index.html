channel 语句

  • 所有log可以输出到一个或多个channel.
  • 每个channel 定义都必须包含一个目标语句，该语句指出为该channel选择的消息是转到file、syslog、stderr，还是被丢弃。
  • 它还可以选择性地限制channel将接受的消息严重性级别（默认为info），以及是否包括命名生成的时间戳，类别名称和/或严重性级别（默认值不包括任何 ）。
