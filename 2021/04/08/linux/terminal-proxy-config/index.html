Linux Terminal Proxy 代理配置

如果需要在 Terminal 中使用代理，则需配置一些代理变量来实现。此方式适用于所有 Linux 发行版、MacOS 等类Unix系统。

通用配置

如果代理启用了身份认证，需要提供账号密码

  • 在当前 BASH 终端中输入可临时生效
  • 将内容写入 ~/.bashrc 中，对当前用户生效
  • 将内容写入 /etc/profile 中，对所有用户生效
  • 在脚本中写入，仅对脚本中的请求生效
